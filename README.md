# News-AlamoFire-SwiftyJSON-iOS

## How to run a news app
Open terminal and write next lines in terminal:
```
cd Path\To\Project
pod install
```

## Pods
Used pods:
- pod 'Alamofire'
- pod 'SwiftyJSON'

<br/>You can also see all used pods in "Podfile".
